智能体热门 Skills 汇总

Claude Code 和 Codex 装好后，下一步就是装技能——但面对散落各处的技能仓库，该装哪些、哪个好用、地址在哪？本文从两大平台通用技能中筛选出高频实用款，按功能维度分类，每款标注可访问地址与核心优势，免去逐一试错的成本。

核心开发辅助类（代码生成、补全、重构）

Superpowers（开发工作流全能工具）

地址：GitHub https://github.com/obra/superpowers

简介：一站式覆盖开发全流程，整合代码生成、调试、文档同步、依赖管理等核心功能，适配各类编程语言与项目规模，帮你摆脱繁琐操作，大幅提升开发效率。

Code Completion（智能代码补全）

地址：GitHub https://github.com/openai/codex/blob/main/docs/skills.md

简介：支持 200+ 编程语言，行内补全、多函数批量补全一键触发，精准适配你的编码风格，告别重复敲码，节省大量开发时间。

Code Refactoring（代码重构）

地址：GitHub https://github.com/anthropics/skills

简介：智能优化代码结构、清除冗余代码、转换编码风格，灵活应用设计模式梳理逻辑，让你的代码更简洁、易维护、可扩展。

写完代码只是第一步，质量管控同样关键。以下技能覆盖审查、测试、安全三大环节，确保代码经得起线上检验。

代码质量管控类（审查、测试、安全）

Code Review（代码审查神器）

地址：GitHub https://github.com/melodic-software/claude-code-plugins/blob/main/plugins/code-quality/skills/review/SKILL.md

简介：前置提交、PR 对比审查双重保障，精准检测安全漏洞、性能瓶颈及代码规范问题，自动生成分级整改建议，守住代码质量底线。

Test Generation（单元测试自动生成）

地址：GitHub https://github.com/openai/codex/blob/main/docs/skills.md

简介：适配主流测试框架，自动生成单元测试、集成测试代码，全面覆盖边缘场景，同步分析代码覆盖率，快速提升代码可靠性，减少调试成本。

Security Audit（安全审计）

地址：GitHub https://github.com/openai/codex/blob/main/docs/skills.md

简介：精准识别 SQL 注入、XSS 等常见安全漏洞，严格遵循 OWASP 规范生成可直接落地的整改方案，全方位守护代码安全。

代码之外，文档处理和项目规划也是日常高频需求。以下技能帮你从手动操作中解放出来。

文档与工具类（文档处理、文件规划）

官方文档四件套（docx/xlsx/pdf/pptx）

地址：官网 https://claude.com/marketplace

简介：无需额外安装工具，一键生成、编辑各类办公文档，实现代码与文档无缝联动，高效完成接口文档、数据统计等文档类工作。

planning-with-files（文件规划技能）

地址：GitHub https://github.com/OthmanAdi/planning-with-files

简介：智能梳理项目文件结构、生成规范目录、规划合理存储方案，告别文件混乱，大幅提升项目管理效率，便于团队协作与后期维护。

通用技能之外，前端开发、算法探索、中文场景各有专属需求。以下专项技能针对这些场景深度优化。

专项辅助类（前端、算法、中文适配）

Frontend-Design（前端审美改造）

地址：GitHub https://github.com/anthropics/skills

简介：前端开发必备神器，智能优化页面布局、统一 CSS 风格，生成符合 UI/UX 规范的 Vue、React 组件代码，快速打造美观且适配的前端页面。

algorithmic-art（算法艺术生成技能）

地址：GitHub https://github.com/openai/codex/blob/main/docs/skills.md

简介：用代码解锁创意，自动生成抽象艺术、数据可视化作品，支持自定义风格与色彩搭配，轻松实现创意开发与视觉呈现。

baoyu-skills（中文适配技能集）

地址：GitHub https://github.com/JimLiu/baoyu-skills

简介：专为中文用户优化，精准识别中文指令，支持中文文档生成、代码注释、问题排查，无需频繁切换中英文，使用更便捷。

专项技能解决特定场景问题，而效率类技能则作用于所有场景——联网检索、提示词优化、循环逻辑改进，用对了能省下大量重复劳动。

效率提升类（联网、提示词、循环优化）

Web-Access（联网搜索+浏览器操控）

地址：GitHub https://github.com/anthropics/skills

简介：打破 AI 训练数据滞后局限，自动联网搜索最新技术文档、框架更新日志，可操控浏览器完成爬取、接口测试，高效助力技术调研。

skill-prompt-generator（技能提示词生成工具）

地址：GitHub https://github.com/anthropics/skills

简介：自动生成精准的技能调用提示词，避免因提示词不规范导致调用失败，大幅提升技能使用效率，新手也能轻松上手。

ralph loop（循环逻辑优化技能）

地址：GitHub https://github.com/openai/codex/blob/main/docs/skills.md

简介：智能检测循环冗余、死循环风险，优化循环逻辑与执行效率，大幅提升代码运行速度，适配各类编程语言的循环场景。

Humanizer（AI 文本去痕技能）

地址：GitHub https://github.com/blader/humanizer

简介：基于 Wikipedia「AI 写作痕迹」指南，自动检测并消除文本中的 AI 生成痕迹（29 种模式），涵盖内容模式（意义夸大、促销语言）、语言模式（AI 高频词汇、同义替换循环）、风格模式（破折号/粗体滥用、表情符号）、沟通模式（聊天机器人痕迹、奉承语气）及填充/模糊模式。支持上传个人写作样本校准输出风格，最终附带「明显 AI 生成」审计扫描确保质量。中文用户可选汉化版 op7418/Humanizer-zh。

如果你想进一步了解技能生态、快速创建自有技能，以下合集与指南类技能是最佳入口。

工具与指南类（技能合集、创建、使用指南）

agent-skills（生产级技能合集）

地址：GitHub https://github.com/addyosmani/agent-skills

简介：由 Addy Osmani 维护的高质量 AI 编程智能体技能合集，提供生产级工程技能，适配 Claude Code、Cursor 等主流智能体，涵盖代码审查、重构、测试等核心场景，是提升开发效率的必备资源库。

everything-claude-code（通用技能合集）

地址：https://github.com/affaan-m/everything-claude-code

简介：整合各类基础开发技能，一键启用无需单独安装，操作简单易上手，是新手入门、快速开启 AI 编程辅助的必备工具包。

awssome-claude-skills（优质技能汇总库）

地址：GitHub https://github.com/ComposioHQ/awesome-claude-skills

简介：社区精选优质技能汇总，附带技能评分与详细教程，可快速筛选适配自身需求的技能，支持社区贡献与实时更新，一站式获取各类实用插件。

claude-howto（智能体使用指南技能）

地址：GitHub https://github.com/anthropics/skills

简介：新手友好型使用指南，快速解答技能安装、指令使用、问题排查等常见疑问，生成个性化 step-by-step 教程，轻松上手 AI 编程助手。

skill-creator（技能创建工具）

地址：GitHub https://github.com/anthropics/skills

简介：无需复杂编码，通过可视化配置即可生成专属技能，精准适配个人个性化需求与团队专属开发场景，灵活拓展 AI 编程辅助能力。

通用使用注意事项

  • 所有 Skills 均需确保 Claude Code/Codex 已启用「代码执行」功能，企业版需管理员开启 Skills 权限。

  • Skills 优先从官方市场获取，社区 Skills 需验证源码安全性，优先选择 Star 数高、持续更新的仓库。

  • Codex 启用 Skills 需放入对应目录，CLI 需升级至 v0.65.0 及以上版本；Claude Code 直接上传压缩包启用。

  • 部分 Skills 需配置 API Key，配置后可提升响应速度与功能完整性。

以上技能均适配 Claude Code 与 OpenAI Codex，按功能分类便于按需取用。建议从核心开发辅助类（Superpowers、Code Refactoring）入手解决日常编码痛点，再按实际需求补充质量管控或效率提升类技能。
