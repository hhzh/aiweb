Codex 教程

Codex 是 OpenAI 推出的终端 AI 编程工具，以任务驱动为核心，支持沙箱执行、多模型切换、MCP 扩展、子代理协作等能力，帮助开发者在命令行中完成代码生成、调试、重构等全流程工作。

本系列教程共 15 篇，从基础配置到高级定制化，系统覆盖 Codex 的核心功能与实战技巧。

入门篇

  • 基础配置教程：安装、API Key 配置、模型选择，快速跑通第一个任务
  • 高级配置教程：权限控制、环境变量、沙箱策略等进阶配置项
  • 提示词使用教程：掌握高效提示词写法，让 Codex 精准理解你的意图
  • Hooks 使用教程：通过钩子机制实现操作拦截与流程自动化

进阶篇

  • MCP 全流程使用教程：接入外部工具与服务，扩展 Codex 能力边界
  • Skills 使用教程：定义可复用的技能模板，标准化常见任务
  • Plugins 使用教程：插件安装、管理与使用
  • Rules 使用教程：通过规则文件约束 Codex 行为，确保输出一致性
  • Subagent 实用教程：拆分复杂任务，子代理独立执行、结果汇总

高级篇

  • 记忆功能使用教程：理解上下文记忆机制，优化长会话体验
  • AGENTS 加载指令详解：深入理解 AGENTS 配置与指令加载流程
  • 官方工作流指南：官方推荐的工作流模式与最佳编排方式
  • 项目定制化教程：按项目需求定制 Codex 的行为、风格与约束
  • 示例配置教程：常见场景的完整配置示例，即抄即用
  • 最佳实践：生产级使用经验汇总，避坑与提效
