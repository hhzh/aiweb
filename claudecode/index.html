Claude Code 教程

Claude Code 是 Anthropic 推出的终端原生 AI 编程工具，支持任务驱动、多模型切换、MCP 扩展、子代理协作等能力，从代码生成到项目交付全流程提效。

本系列教程从零基础上手到高级功能深入，共 14 篇，覆盖安装配置、权限管理、工作流、Hooks、MCP、Skills、子代理、内存与目录配置、扩展与插件、最佳实践等核心主题，助你全面掌握 Claude Code。

入门篇

  • 快速上手指南：30 分钟完成安装、登录、配置，新手必读
  • 官方工作原理与使用指南：理解 Claude Code 的架构设计、运行机制与核心概念
  • 权限模式完全指南：掌握 Plan/Auto/Yolo 等权限模式，安全高效地使用工具
  • 常见工作流：代码生成、Bug 修复、重构、项目初始化等典型场景实操

进阶篇

  • 斜杠命令实战教程：10+ 高频命令详解，提升日常操作效率
  • Hooks 实战教程：通过钩子机制实现操作自动化与流程编排
  • MCP 使用教程：接入外部工具与服务，扩展 Claude Code 能力边界
  • Skills 使用教程：定义可复用的技能模板，标准化常见任务

高级篇

  • 子代理使用指南：拆分复杂任务，主从协同，并行处理专项工作
  • 内存配置文档：理解上下文管理与内存优化策略
  • .claude 目录配置指南：项目级与全局配置详解，精细化控制行为
  • 扩展选型与配置手册：IDE 集成、模型切换、第三方扩展选型
  • 插件使用教程：插件安装、管理与自定义开发
  • 最佳实践：生产级使用经验汇总，避坑与提效
