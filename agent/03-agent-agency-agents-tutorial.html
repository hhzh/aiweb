Agency Agents：AI 专业智能体合集全面解析

Agency Agents：AI 专业智能体合集全面解析

项目官网：https://github.com/msitarzewski/agency-agents

一个人做产品，缺设计师做视觉、缺安全工程师做审计、缺运营写文案——agency-agents 把 140+ 专业智能体封装为开箱即用的角色，覆盖 12 大业务部门，适配 Claude Code、GitHub Copilot、Cursor 等主流 AI 工具，让个人与小团队瞬间获得企业级的专业能力支撑。

一、项目核心概览

agency-agents 是一款开源、可定制、生产就绪的 AI 专业智能体库，源自社区实践与长期迭代，采用 MIT 开源协议免费使用。

核心定位

把 AI 从 “通用助手” 升级为细分领域专家，每个智能体都具备：

  • 专业化能力：深耕单一领域，而非泛化提示词模板

  • 人格化特质：拥有专属沟通风格与工作逻辑

  • 可交付成果：输出可落地的代码、方案、报告、流程

  • 生产级标准：经过实战验证，自带验收指标与工作流

核心数据

  • 140+专业智能体，覆盖 12 大业务部门

  • 10000 + 行专业工作流、代码示例与交付规范

  • 适配10 + 主流 AI 编码 / 协作工具

  • 社区驱动迭代，持续新增领域与优化能力

二、30 秒快速上手

了解核心定位后，直接安装上手。

环境要求

  • 支持 macOS/Linux/Windows（Git Bash/WSL）

  • 基础 Shell 执行能力

  • 目标 AI 工具（如 Claude Code、Copilot、Cursor）

一键安装（推荐）

# 克隆仓库
git clone https://github.com/msitarzewski/agency-agents.git
cd agency-agents

# 生成全工具适配文件
./scripts/convert.sh

# 交互式安装（自动检测本地工具）
./scripts/install.sh

极简安装（仅 Claude Code）

./scripts/install.sh --tool claude-code

安装完成后，在 Claude Code 中直接调用：

使用前端开发者智能体帮我构建React组件

三、12 大部门智能体全解

agency-agents 按业务场景划分为 12 大部门，覆盖技术研发、创意设计、市场营销、产品管理、游戏开发、学术研究等全场景，按需取用。

1. 工程研发部（核心技术岗）

面向全栈开发、架构、运维、安全等技术场景，是开发者的最强辅助：

  • 前端开发者：React/Vue/Angular、UI 实现、性能优化

  • 后端架构师：API 设计、数据库架构、微服务

  • AI 工程师：模型部署、LLM 集成、数据 pipeline

  • 安全工程师：威胁建模、代码审计、漏洞检测

  • DevOps 自动化专家：CI/CD、云基础设施、监控告警

2. 创意设计部

专注视觉、体验、品牌与创意产出：

  • UI/UX 设计师：设计系统、交互流程、视觉规范

  • 品牌守护者：品牌定位、视觉统一、风格把控

  • AI 图像提示工程师：Midjourney/DALL-E 精准提示词

3. 市场营销部

覆盖全球主流平台与增长玩法，适配国内小红书、抖音、B 站、微信生态：

  • 内容创作者：全平台文案、选题、 editorial 日历

  • 小红书 / 抖音 / 微信专家：平台算法、流量玩法、内容运营

  • SEO/ASO 优化师：搜索排名、应用商店增长

  • 跨境电商专家：亚马逊、Shopee 全流程运营

4. 销售与付费媒体部

助力获客、转化、营收增长：

  • outbound 策略师：潜客挖掘、多渠道触达

  • PPC 投放专家：谷歌 / Meta 广告、预算优化

  • 销售工程师：技术演示、POC 方案、竞争分析

5. 产品与项目部

产品规划、项目管控、需求落地：

  • 产品经理：PRD、路线图、需求梳理

  • 敏捷冲刺规划师：优先级排序、backlog 管理

  • 高级项目经理：范围管控、任务拆解、进度协调

6. 测试与质量保障部

全维度质量把关，提前规避线上问题：

  • UI 视觉测试员：截图验证、界面一致性检查

  • 性能基准测试：页面速度、负载、资源占用

  • 无障碍审计：WCAG 合规、辅助设备适配

7. 游戏开发部

跨引擎支持，覆盖 Unity/Unreal/Godot/Roblox：

  • 游戏设计师：GDD、玩法循环、经济平衡

  • 引擎工程师：各引擎专属开发、网络同步、Shader 编写

  • 技术美术：特效、材质、资源优化

8. 金融财务部

会计、财务分析、税务、投资研究：

  • 簿记 / 财务主管：月末结账、GAAP 合规、对账

  • 财务分析师：建模、预测、场景分析

  • 税务策略师：多地区合规、税负优化

9. 空间计算部

面向 AR/VR/visionOS 沉浸式开发：

  • XR 界面架构师：空间交互、沉浸式体验

  • visionOS 工程师：Apple Vision Pro 原生开发

10. 学术研究部

为叙事、世界观、角色设计提供学术支撑：

  • 人类学家 / 地理学家 / 历史学家：文化、地理、历史真实性

  • 叙事学家 / 心理学家：故事结构、角色动机、情感逻辑

11. 专项特种部

覆盖法律、合规、招聘、供应链、政企服务等小众刚需：

  • 合规审计师：SOC2、ISO27001、HIPAA 认证

  • 招聘专家：人才寻访、面试流程、入职管理

  • 政企售前顾问：政府项目方案、数字化转型

12. 客户支持部

全行业客户服务、数据分析、 executive 汇报：

  • 全渠道客服：零售、SaaS、酒店、医疗合规应答

  • 数据分析师：仪表盘、KPI 追踪、业务洞察

四、多 AI 工具无缝集成

agency-agents 最大优势是一次编写，全工具运行，自动适配主流 AI 协作环境：

工具安装命令调用方式
Claude Code./scripts/install.sh --tool claude-code直接描述调用
GitHub Copilot./scripts/install.sh --tool copilot指令式调用
Cursor./scripts/install.sh --tool cursor规则自动生效
OpenClaw./scripts/install.sh --tool openclaw工作区加载
Gemini CLI./scripts/install.sh --tool gemini-cli扩展指令调用
Kimi Code./scripts/install.sh --tool kimi配置文件加载

集成核心优势

  • 无重复配置：自动生成各工具专属格式

  • 无侵入性：不修改原有工作流

  • 自动检测：安装器扫描本地工具，一键启用

五、真实场景案例

了解了 12 大部门的智能体后，以下案例展示如何组合多个智能体完成端到端任务。

案例 1：初创产品 MVP 快速交付

  1. 产品经理智能体：输出 PRD 与功能清单

  2. 前端 + 后端智能体：并行开发界面与 API

  3. 测试智能体：自动化验证与 bug 修复

  4. 增长黑客智能体：制定冷启动获客方案
    结果：单人 3 天完成传统 2 周工作量

案例 2：全渠道营销 campaign

  1. 内容创作者：生成全平台文案

  2. 小红书 / 抖音 / B 站智能体：平台专属内容优化

  3. 数据分析智能体：效果追踪与迭代建议
    结果：多平台内容统一调性，精准触达目标用户

案例 3：企业级功能迭代

  1. 项目经理：拆解任务与排期

  2. senior 开发者：核心逻辑实现

  3. UI 设计师：界面与交互优化

  4. 质量审计：上线前验收把关
    结果：流程规范，质量可控，上线零故障

案例 4：付费媒体账户接管与优化

  1. 付费媒体审计师：对现有广告账户进行200+项全面审计，排查浪费性支出

  2. 搜索查询分析师：清理无效搜索词，优化关键词策略，减少无效点击

  3. PPC投放专家：重构账户架构，优化出价策略，提升广告ROI

  4. 广告创意策略师：刷新广告文案与素材，缓解广告疲劳

  5. 数据分析智能体：搭建实时监控仪表盘，跟踪投放效果并迭代
    结果：30天内广告浪费支出减少40%，广告转化率提升25%，账户整体ROI提升35%

案例 5：小型游戏原型开发（Unity引擎）

  1. 游戏设计师智能体：输出完整游戏设计文档（GDD），明确玩法循环与经济系统

  2. Unity架构师：设计游戏核心框架，搭建数据驱动模块

  3. 技术美术智能体：编写基础Shader，优化场景资源，保障运行流畅度

  4. 测试智能体：自动化检测游戏BUG，验证玩法逻辑与性能表现
    结果：单人2周完成Unity游戏原型开发，涵盖核心玩法、视觉呈现与基础交互，无需专业游戏开发团队支撑

案例 6：学术向世界观构建（游戏/影视场景）

  1. 历史学家智能体：为世界观设定符合历史逻辑的时代背景与社会结构

  2. 地理学家智能体：设计地理环境、气候特征，确保场景设定的合理性

  3. 人类学家智能体：构建独特的文化、礼仪与信仰体系，丰富世界观细节

  4. 叙事学家智能体：结合设定，梳理故事主线与角色动机，提升叙事连贯性
    结果：快速输出逻辑自洽、细节丰富的完整世界观体系，为内容创作提供坚实学术支撑，节省60%的设定时间

六、定制与贡献指南

自定义智能体

  1. Fork 仓库

  2. 在对应部门目录新建.md智能体文件

  3. 遵循模板：身份→核心任务→工作流→交付标准→验收指标

  4. 执行./scripts/convert.sh重新生成适配文件

贡献流程

  • 新增智能体：提交 PR 到对应分类

  • 优化能力：补充代码示例、工作流、成功指标

  • 本地化：新增中文场景适配（如微信、小红书、抖音）

七、总结

agency-agents 的核心价值是"一次编写，全工具运行"——140+ 专业智能体按 12 大部门分类，适配 10+ 主流 AI 工具，无需重复配置。建议先从工程研发部的智能体入手（如前端开发者、安全工程师），体验单一智能体的专业能力，再尝试多智能体组合完成端到端任务（如产品经理 → 开发者 → 测试员的 MVP 交付流程）。
