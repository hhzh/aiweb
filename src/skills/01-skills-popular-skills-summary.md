# 智能体热门 Skills 汇总

Claude Code 和 Codex 装好后，下一步就是装技能——但面对散落各处的技能仓库，该装哪些、哪个好用、地址在哪？本文从两大平台通用技能中筛选出高频实用款，按功能维度分类，每款标注可访问地址与核心优势，免去逐一试错的成本。

## 核心开发辅助类（代码生成、补全、重构）

### Superpowers（开发工作流全能工具）

**地址**：GitHub https://github.com/obra/superpowers

**简介**：一站式覆盖开发全流程，整合代码生成、调试、文档同步、依赖管理等核心功能，适配各类编程语言与项目规模，帮你摆脱繁琐操作，大幅提升开发效率。

### Code Completion（智能代码补全）

**地址**：GitHub https://github.com/openai/codex/blob/main/docs/skills.md

**简介**：支持 200+ 编程语言，行内补全、多函数批量补全一键触发，精准适配你的编码风格，告别重复敲码，节省大量开发时间。

### Code Refactoring（代码重构）

**地址**：GitHub https://github.com/anthropics/skills

**简介**：智能优化代码结构、清除冗余代码、转换编码风格，灵活应用设计模式梳理逻辑，让你的代码更简洁、易维护、可扩展。

写完代码只是第一步，质量管控同样关键。以下技能覆盖审查、测试、安全三大环节，确保代码经得起线上检验。

## 代码质量管控类（审查、测试、安全）

### Code Review（代码审查神器）

**地址**：GitHub https://github.com/melodic-software/claude-code-plugins/blob/main/plugins/code-quality/skills/review/SKILL.md

**简介**：前置提交、PR 对比审查双重保障，精准检测安全漏洞、性能瓶颈及代码规范问题，自动生成分级整改建议，守住代码质量底线。

### Test Generation（单元测试自动生成）

**地址**：GitHub https://github.com/openai/codex/blob/main/docs/skills.md

**简介**：适配主流测试框架，自动生成单元测试、集成测试代码，全面覆盖边缘场景，同步分析代码覆盖率，快速提升代码可靠性，减少调试成本。

### Security Audit（安全审计）

**地址**：GitHub https://github.com/openai/codex/blob/main/docs/skills.md

**简介**：精准识别 SQL 注入、XSS 等常见安全漏洞，严格遵循 OWASP 规范生成可直接落地的整改方案，全方位守护代码安全。

## 通用使用注意事项

- 所有 Skills 均需确保 Claude Code/Codex 已启用「代码执行」功能，企业版需管理员开启 Skills 权限。

- Skills 优先从官方市场获取，社区 Skills 需验证源码安全性，优先选择 Star 数高、持续更新的仓库。

- Codex 启用 Skills 需放入对应目录，CLI 需升级至 v0.65.0 及以上版本；Claude Code 直接上传压缩包启用。

- 部分 Skills 需配置 API Key，配置后可提升响应速度与功能完整性。

以上技能均适配 Claude Code 与 OpenAI Codex，按功能分类便于按需取用。建议从核心开发辅助类（Superpowers、Code Refactoring）入手解决日常编码痛点，再按实际需求补充质量管控或效率提升类技能。
