# OpenCode 教程

OpenCode 是一款开源 AI 编码代理（AI Coding Agent），主打终端优先的交互模式，同时提供桌面应用、IDE 扩展等多端使用方案。它支持对接市面上主流大模型（Claude、GPT、Gemini 等）及本地离线模型，无厂商锁定，且遵循隐私优先设计。

本系列教程共 10 篇，从零基础上手到高级功能深入，覆盖安装配置、命令使用、工具权限、Skill 技能、插件开发、模型切换、Rule 规则、代理配置、UI 界面与 IDE 集成等核心主题，助你全面掌握 OpenCode。

## 入门篇

- **快速入门教程**：30 分钟从安装到第一个项目，新手必读
- **命令使用教程**：详解 CLI 命令、TUI 操作、自定义命令与快捷键
- **工具使用教程**：内置工具、自定义工具、MCP 扩展与格式化工具

## 进阶篇

- **Skills 使用教程**：定义可复用的技能文件，标准化常见任务
- **插件使用教程**：插件安装、事件钩子与自定义插件开发
- **切换模型使用教程**：对接 75+ 模型提供商，配置模型变体与参数

## 高级篇

- **Rule 使用教程**：用 AGENTS.md 定义项目规则，统一 AI 行为
- **权限配置教程**：精细化权限管控，平衡效率与安全
- **Open Agent 使用教程**：自定义代理，实现任务分工与权限隔离
- **UI 界面与 IDE 集成教程**：TUI、Web 界面与 VS Code/Cursor 集成
