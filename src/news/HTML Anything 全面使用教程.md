# HTML Anything 使用教程

## 一、工具概述

**HTML Anything** 是一款面向智能体时代的轻量化 HTML 编辑器，由`nexu\-io`团队打造（开源仓库：[https://github\.com/nexu\-io/html\-anything](https://github.com/nexu-io/html-anything)）。它彻底改变传统文档编辑方式 ——**Markdown 是草稿，HTML 才是最终呈现形态**，依托本地 AI 能力，一键将各类内容生成可直接发布的精美 HTML，无需手动写代码、调样式。

### 核心价值

告别手动编写 CSS、调试布局的繁琐，专注内容创作；生成的 HTML 兼顾美观与实用性，可直接用于公众号、知乎、小红书、网页原型等多场景，无需二次排版优化。

### 核心优势

- **本地优先・零 API 密钥**：复用本地已登录的 AI 命令行工具会话，无需额外申请、配置 API Key，降低使用门槛；

- **全 AI 工具兼容**：自动识别并适配 8 大主流编码 AI CLI（Claude Code、Codex、Cursor Agent、Gemini CLI、GitHub Copilot CLI、OpenCode、Qwen Coder、Aider）；

- **75 \+ 精美模板**：覆盖 9 大场景（杂志文章、演示文稿、简历、海报、小红书卡片、推文卡片、网页原型、数据报表、视频帧），开箱即用；

- **一键多端导出**：支持导出 HTML、高清 PNG，可直接粘贴到微信、知乎、X（推特）等平台，自动适配平台格式；

- **流式实时渲染**：AI 生成过程实时预览，可随时中断重写，无需等待完整生成；

- **沙箱安全预览**：生成的 HTML 在隔离沙箱中运行，保障本地环境安全。

## 二、安装部署（3 步快速上手）

### 1\. 环境准备

- 系统要求：Windows、macOS、Linux 全平台兼容；

- 依赖工具：安装`Node\.js 18\+`、`pnpm`（包管理器）；

- 前置条件：本地已安装并登录任意一款支持的 AI CLI（如 Claude Code、Gemini CLI）。

### 2\. 克隆仓库

```bash
# 克隆HTML Anything仓库
git clone https://github.com/nexu-io/html-anything

# 进入项目目录
cd html-anything
```

### 3\. 安装依赖并启动

```bash
# 安装项目依赖
pnpm install

# 启动开发服务
pnpm -F @html-anything/next dev
```

启动成功后，浏览器自动打开`http://localhost:3000`，即可进入 HTML Anything 编辑界面。

## 三、界面总览

HTML Anything 界面简洁直观，分为 5 大核心区域，新手可快速上手：

- **顶部工具栏**：自动识别已登录的 AI CLI（可切换）、导出按钮、设置按钮；

- **左侧编辑区**：输入 / 粘贴内容（支持 Markdown、CSV、JSON、纯文本等多格式）；

- **中间模板区**：按场景筛选模板（演示文稿、网页、海报等），支持搜索；

- **右侧预览区**：实时显示生成的 HTML 效果，沙箱隔离运行；

- **底部状态栏**：显示生成进度、模板信息。

## 四、核心使用步骤（4 步生成精美 HTML）

### 步骤 1：选择 AI 工具

启动后，顶部工具栏自动扫描本地已登录的 AI CLI：

- 若已安装多款，点击下拉框切换（如 Claude Code、Gemini CLI）；

- 若未识别，确认 AI CLI 已添加到系统`PATH`，且完成登录。

### 步骤 2：输入创作内容

左侧编辑区支持**多格式内容输入**，直接粘贴或手动输入即可：

- 支持格式：Markdown、纯文本、CSV、Excel、JSON、SQL；

- 示例：输入一篇产品文案、技术文档、小红书笔记，无需格式化；

- 支持多段内容、图片描述（生成时自动适配图片布局）。

### 步骤 3：选择模板（关键）

中间模板区提供**75 \+ 专业模板**，按 9 大场景分类：

#### 热门精选模板

- **演示文稿类**：`deck\-swiss\-international`（瑞士风极简）、`deck\-guizang\-editorial`（杂志风）；

- **文档类**：`doc\-kami\-parchment`（暖羊皮纸风，适合长文）；

- **社交媒体类**：`card\-xiaohongshu`（小红书卡片）、`social\-x\-post\-card`（推特卡片）；

- **网页原型类**：`saas\-landing`（SaaS 落地页）、`dashboard`（数据看板）；

- **视频帧类**：`frame\-glitch\-title`（故障风标题）、`video\-hyperframes`（动态视频帧）。

#### 筛选技巧

- 按场景筛选：点击`deck`（演示）、`social`（社交）、`prototype`（网页）等标签；

- 搜索模板：输入关键词（如 “小红书”“简历”）快速匹配；

- 预览模板：点击模板缩略图，右侧实时预览效果。

### 步骤 4：生成并导出 HTML

#### 1\. 生成内容

确认内容和模板后，按`⌘\+Enter`（macOS）或`Ctrl\+Enter`（Windows），AI 开始流式生成 HTML：

- 右侧预览区**实时渲染**，可随时点击 “中断” 重写；

- 生成过程显示进度，完成后自动适配模板样式。

#### 2\. 导出发布

生成完成后，点击顶部**导出按钮**，支持 5 种导出方式：

- **复制到剪贴板**：直接粘贴到微信公众号、知乎、X，自动适配平台格式；

- **下载 HTML**：保存本地，可直接部署到网页服务器；

- **下载 PNG**：导出高清图片，适合小红书、朋友圈；

- **微信适配**：自动内联 CSS，粘贴到微信公众号无格式错乱；

- **知乎适配**：自动处理公式、排版，直接发布。

## 五、高级功能

### 1\. 流式实时编辑

生成过程中，右侧预览区**逐行渲染**HTML：

- 可随时中断，修改左侧内容后重新生成；

- 支持实时调整模板样式，无需重新生成。

### 2\. 模板自定义

- 模板硬约束：所有模板内置**排版规范**（8px 基线网格、CJK 字体优先、对比度≥4\.5），避免 AI 生成混乱样式；

- 自定义模板：进阶用户可在`next/src/lib/templates/skills`目录添加模板（遵循`SKILL\.md`规范）。

### 3\. 多格式适配

- 自动识别输入格式：Markdown、CSV、JSON 等自动适配排版；

- 表格优化：CSV/Excel 自动生成美观表格，支持排序、高亮；

- 代码格式化：Markdown 代码块自动高亮、换行。

## 六、常见问题

### 1\. 启动后未识别 AI CLI

- 排查：确认 AI CLI 已添加到系统`PATH`（如`\~/\.local/bin`、`/opt/homebrew/bin`）；

- 解决：重启终端，重新登录 AI CLI，再启动 HTML Anything。

### 2\. 生成样式混乱

- 原因：输入内容格式不规范，或模板选择不匹配；

- 解决：选择匹配场景的模板，输入内容避免过多特殊符号。

### 3\. 导出微信格式错乱

- 原因：未使用 “微信适配” 导出选项；

- 解决：导出时选择 “微信适配”，自动内联 CSS。

### 4\. 模板太少

- 解决：关注仓库更新，或自定义模板提交 PR。

无需懂代码，输入内容、选模板、一键生成，即可产出专业级 HTML，覆盖文档、演示、社交、网页等全场景。适合内容创作者、产品经理、程序员、设计师等各类人群，大幅提升创作效率。
